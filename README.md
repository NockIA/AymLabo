# AymLabo

Aymlabo is an Aimlab be like, in other words an aiming training game.

## Start the project
`git clone https://github.com/NockIA/AymLabo.git`

### Terminal 1 :
`cd Api`
`go run main.go --init`
`go run main.go`

### Terminal 2 :
`cd Application/`
`npm i`

> To launch dynamically :

- `npm run dev`

> To launch the software :

 - npm run build
 - go to the release/1.0.0 folder 
 - install the software  
 - run the software
 
 ## Features
 ### Game :
 - Gridshot (Close targets appearing)
 - Tracking (Moving target)
 ### User :
 - Authentication
 - Leaderboard
	 - filter by friends
	 - filter by categories
	 - pagination
 - Profile / Stats
	 - personalise avatar
	 - change informations
	 - display personal stats
